# SOC Investigations Portfolio

This repository contains cybersecurity investigation exercises completed through SOC simulation platforms including LetsDefend, CyberDefenders, BTLO, and BOSS OF THE SOC (BOTS).

The purpose of this repository is to document my learning journey in Security Operations Center (SOC) analysis, incident investigation, log analysis, and network traffic analysis.

## Skills Practiced

* Alert Triage
* Log Analysis
* Incident Investigation
* Security Monitoring
* Network Traffic Analysis
* Threat Detection
* MITRE ATT&CK Mapping
* Evidence Collection
* IOC Identification
