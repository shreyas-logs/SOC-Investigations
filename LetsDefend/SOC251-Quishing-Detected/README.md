# SOC251 - Quishing Detected (QR Code Phishing)

## Objective

Investigate a suspected QR code phishing attack and determine whether the alert represents malicious activity.

## Alert Summary

A security alert was generated indicating a potentially malicious QR code embedded within an email.

## Investigation Steps

1. Reviewed alert details.
2. Examined sender information.
3. Analyzed embedded QR code destination.
4. Identified suspicious indicators.
5. Validated findings using investigation workflow.

## Findings

* Suspicious QR code identified.
* Destination URL determined to be malicious.
* Alert classified as a phishing attempt.

## MITRE ATT&CK

* T1566 - Phishing

## Lessons Learned

This investigation improved understanding of phishing detection techniques and the importance of validating embedded URLs within email communications.
